Summary: Latitude vs. X

Visualizations

The purpose of this project is to examine the change in weather across a variety of locations using their relative difference is distance from the equator as a reference. Using Openweather API we were able to examine over 500 cities and their weather conditions at that moment.

After this information was collected it needed to be processed which required information to cleaned and then displayed using matplotlib's graphing functions. The variables examined against the latitude were the cloudiness, temperature, wind speed and humidty. This site will be used to display and convey our findings on this information.
